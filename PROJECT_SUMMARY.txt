How to Use
==========

INSTALLATION (5 minutes):
  1. Install Node.js from nodejs.org
  2. Open Command Prompt
  3. Navigate: cd /d "c:\Users\User\Desktop\ieltspro2\vs code"
  4. Install: npm install
  5. Start: npm start
  6. Open: http://localhost:3000

AS STUDENT (Free browsing):
  1. View all 4 modules
  2. Access listening audio

  5. No authentication needed

AS ADMIN (With PIN 2010):
