IELTS PREPARATION PLATFORM - COMPLETE INSTALLATION & USAGE GUIDE
==================================================================

PROJECT OVERVIEW
================
A fully functional website for IELTS preparation with:
✅ Four learning modules (Listening, Reading, Writing, Speaking)
✅ Admin panel with file upload capabilities  
✅ Audio player for listening practice
✅ Material management system
✅ Professional responsive design

SYSTEM REQUIREMENTS
===================
- Windows 10/11 or Mac/Linux
- Node.js v14+ (Download from nodejs.org)
- npm (included with Node.js)
- Modern web browser (Chrome, Firefox, Edge, Safari)
- ~200MB free disk space

INSTALLATION STEPS
==================

STEP 1: Install Node.js
-----------------------
1. Download Node.js from: https://nodejs.org
2. Choose LTS (Long Term Support) version
3. Run the installer
4. Accept default settings
5. Verify installation:
   - Open Command Prompt
   - Type: node --version
   - You should see version number (e.g., v16.13.0)

STEP 2: Navigate to Project Directory
--------------------------------------
1. Open Command Prompt (Windows) or Terminal (Mac/Linux)
2. Type the following command:

   cd /d "c:\Users\User\Desktop\ieltspro2\vs code"

   (Note: /d flag allows changing drives on Windows)

STEP 3: Install Dependencies
-----------------------------
1. In Command Prompt/Terminal, type:

   npm install

2. Wait for installation to complete (1-3 minutes)
3. You should see "added XX packages"

STEP 4: Start the Server
------------------------
1. In Command Prompt/Terminal, type:

   npm start

2. You should see:
   "IELTS Preparation Platform running on http://localhost:3000"

3. The terminal will keep running - don't close it!

STEP 5: Access the Application
-------------------------------
1. Open your web browser
2. Go to: http://localhost:3000
3. You should see the IELTS Preparation homepage

HOW TO CREATE YOUR OWN MATERIALS
================================

Reading Materials (.html)
--------------------------
Create a simple HTML file with reading passage and questions:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Reading Material</title>
    <style>
        body { font-family: Arial; line-height: 1.6; padding: 20px; }
    </style>
</head>
<body>
    <h1>Reading Passage</h1>
    <p>Your reading text here...</p>
    
    <h2>Questions</h2>
    <p>1. Your question here?</p>
    <p>a) Option A</p>
    <p>b) Option B</p>
</body>
</html>
```

Save as: my-reading.html
Upload via Admin Panel → Reading tab

Audio Files (Listening)
------------------------
- Use any standard audio file format:
  * MP3 (recommended)
  * WAV
  * OGG
  * FLAC

No conversion needed! Just upload directly.

------------------------------
Create professional HTML with tips, examples, and formatting.

TROUBLESHOOTING
===============

Issue: "npm: command not found"
Solution: Node.js not installed. Download from nodejs.org and install.

Issue: "Port 3000 is already in use"
Solution: 
  - Another program is using port 3000
  - Option 1: Close other servers
  - Option 2: Edit server.js, change PORT = 3000 to PORT = 3001

Issue: "Cannot find module 'express'"
Solution: Run "npm install" again in the project directory

Issue: Server starts but browser shows error
Solution:
  - Refresh the page (Ctrl+R or Cmd+R)
  - Check console (F12) for errors
  - Verify http://localhost:3000 (not https)

Issue: Admin code doesn't work
Solution: Default code is "2010" - check for typos

Issue: File uploads fail
Solution:
  - Check file size (keep under 100MB)
  - Verify file format (HTML for text, MP3/WAV/OGG for audio)
  - Ensure title field is not empty
  - Check browser console (F12) for error messages

Issue: Audio player shows but no sound
Solution:
  - Verify audio file is valid (try opening in media player)
  - Try different browser
  - Check browser volume settings
  - MP3 format recommended

Issue: HTML materials don't display correctly
Solution:
  - Create simple HTML without external resources
  - Use inline CSS (put <style> in <head>)
  - Test HTML file locally first before uploading

STOPPING THE SERVER
===================

To stop the server:
1. Go to Command Prompt/Terminal
2. Press: Ctrl + C
3. Type: Y (if prompted)
4. Press: Enter

The server will stop and you'll see the prompt again.

API ENDPOINTS (For Developers)
==============================

Authentication:
POST /api/admin/login
Body: { "code": "2010" }

Upload File:
POST /api/upload/:module
Headers: multipart/form-data
Fields: title, file
Modules: listening, reading, writing, speaking

Get Materials:
GET /api/materials/:module
Returns: { success, materials[] }

Get File:
GET /api/file/:module/:id
Returns: File download/stream

Delete Material:
DELETE /api/materials/:module/:id
Returns: { success, message }
