IELTS PREPARATION PLATFORM - INSTALLATION GUIDE
==================================================

OVERVIEW
--------
Complete setup guide for installing and deploying the IELTS Preparation Platform.

SYSTEM REQUIREMENTS
==================
- Windows 10/11 (or Mac/Linux)
- Node.js v14+ (https://nodejs.org)
- npm (included with Node.js)
- Modern web browser
- ~200MB free disk space

STEP 1: Install Node.js
-----------------------
1. Go to: https://nodejs.org
2. Download LTS (Long Term Support) version
3. Run the installer
4. Accept all default settings
5. Restart your computer (recommended)
6. Verify installation:
   - Open Command Prompt
   - Type: node --version
   - You should see version number (e.g., v16.13.0)

STEP 2: Navigate to Project Directory
--------------------------------------
1. Open Command Prompt
2. Type: cd /d "c:\Users\User\Desktop\ieltspro2\vs code"
3. Press Enter

STEP 3: Install Dependencies
----------------------------
1. Type: npm install
2. Wait for installation (1-3 minutes)
3. You should see: "added XX packages"

STEP 4: Start the Server
-----------------------
1. Type: npm start
2. You should see: "IELTS Preparation Platform running on http://localhost:3000"
3. Terminal will stay running (don't close it!)

STEP 5: Access the Application
-------------------------------
1. Open web browser
2. Go to: http://localhost:3000
3. You should see the IELTS homepage

CUSTOMIZATION
=============

Change Admin Code:
- Edit: server.js
- Find: if (code === '2010')
- Change: '2010' to your desired code

Change Port:
- Edit: server.js  
- Find: const PORT = 3000
- Change: 3000 to any number (3001, 8080, etc.)

Change Colors:
- Edit: style.css or admin-style.css
- Modify CSS variables at top of file
- All colors use --primary-color, --secondary-color, etc.

DEPLOYMENT
==========

Local Network:
1. Find your IP address (ipconfig in Command Prompt)
2. Share IP with others on network
3. They access: http://YOUR_IP:3000

Cloud Deployment (Heroku):
1. Create Heroku account
2. Install Heroku CLI
3. heroku create
4. git push heroku main
5. Get URL and share

Could Deployment (DigitalOcean):
1. Create droplet (Ubuntu 20.04)
2. SSH into droplet
3. Install Node.js and npm
4. Clone repository
5. Run: npm install && npm start
6. Use process manager (PM2) to keep running
7. Set up domain name
8. Configure SSL certificate

TROUBLESHOOTING
===============

Problem: npm not found
Solution: Node.js not installed. Download from nodejs.org

Problem: Port 3000 in use
Solution: 
  - Find what's using port 3000: netstat -ano | findstr :3000
  - Kill process: taskkill /PID <PID> /F
  - Or change port in server.js

Problem: Cannot find module 'express'
Solution: Run: npm install

Problem: Server starts but browser shows error
Solution:
  - Refresh page (Ctrl+R)
  - Check console (F12)
  - Use http:// not https://

Problem: Admin code doesn't work
Solution: Default code is 2010 - check for typos

Problem: File upload fails
Solution:
  - File size under 100MB?
  - File format correct (HTML, MP3, WAV, OGG)?
  - Title entered?
  - Check browser console (F12)

Problem: Audio player shows but no sound
Solution:
  - Try MP3 format (most compatible)
  - Try different browser
  - Check volume settings
  - Verify audio file is valid

Problem: HTML materials don't display
Solution:
  - Create simple HTML without external resources
  - Use inline CSS
  - Test HTML locally first
  - Avoid external scripts

STOPPING THE SERVER
===================
In Command Prompt:
Press: Ctrl + C
Type: Y (if prompted)
Press: Enter
