═══════════════════════════════════════════════════════════════════════════════
                    🛠️ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "npm: command not found"
  → Node.js not installed. Download from nodejs.org

Issue: "Port 3000 already in use"
  → Edit server.js: Change "const PORT = 3000" to "const PORT = 3001"

Issue: Server starts but browser shows error
  → Try http://localhost:3000 (not https)
  → Refresh page (Ctrl+R)
  → Check browser console (F12)

Issue: Admin code doesn't work
  → Code is: 2010 (not 2020)
  → Check for spaces or typos

Issue: File upload fails
  → Check file size (keep under 100MB)
  → For audio: use MP3, WAV, or OGG format
  → For materials: use HTML files
  → Check browser console (F12) for errors

For more help → See SETUP_GUIDE.txt
